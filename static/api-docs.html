📚 x-download API 文档

基于 yt-dlp 的网页版视频下载工具 — 完整 API 参考

Base URL: http://77.73.14.88:8080
← 返回首页
🎬 视频下载核心接口
POST /api/info
获取视频基本信息，包括标题、时长、封面、上传者等。
请求参数
{
  "url": "https://www.youtube.com/watch?v=xxx"
}
返回示例
{
  "success": true,
  "title": "视频标题",
  "duration": 120,
  "thumbnail": "https://...",
  "uploader": "上传者名称"
}
提示：如果返回 success: false，说明视频链接无效或平台不支持。
POST /api/formats
获取视频可用的画质和格式列表。
请求参数
{
  "url": "https://www.youtube.com/watch?v=xxx"
}
返回示例
{
  "success": true,
  "formats": [
    {"id": "best", "desc": "最佳画质（推荐）"},
    {"id": "137+140", "desc": "1080p+音频"}
  ]
}
POST /api/download
下载视频到服务器（同步阻塞，大文件需等待）。
请求参数
{
  "url": "https://www.youtube.com/watch?v=xxx",
  "format": "格式ID（可选，不填则下载最佳画质）"
}
返回示例
{
  "success": true,
  "file": "视频标题_abc12345.mp4",
  "size": 1024000
}
超时 300 秒。文件保存在 /opt/x-download/downloads，超过 MAX_FILES（默认 50）自动清理旧文件。
POST /api/audio
提取视频音频并转换为 MP3 格式。
请求参数
{
  "url": "https://www.youtube.com/watch?v=xxx",
  "format": "bestaudio/best（可选）"
}
返回示例
{
  "success": true,
  "file": "视频标题_abc12345.mp3",
  "size": 1024000
}
自动提取最佳音质，转换为 MP3，比特率 320k。
🔍 搜索与发现
POST /api/search
基于关键词搜索视频（默认返回前 5 条结果）。
请求参数
{
  "query": "关键词"
}
返回示例
{
  "success": true,
  "videos": [
    {
      "id": "dQw4w9WgXcQ",
      "title": "视频标题",
      "url": "https://www.youtube.com/watch?v=...",
      "thumbnail": "https://...",
      "duration": 212,
      "uploader": "频道名称"
    }
  ]
}
基于 yt-dlp 内置搜索，默认搜索 YouTube。可修改代码中的 ytsearch5: 为 ytsearch10: 等调整结果数。
POST /api/check
检查链接是否支持下载，返回基本信息。
请求参数
{
  "url": "https://..."
}
返回示例（支持）
{
  "success": true,
  "supported": true,
  "title": "视频标题",
  "extractor": "youtube",
  "uploader": "频道名称"
}
返回示例（不支持）
{
  "success": true,
  "supported": false,
  "error": "不支持此链接"
}
GET /api/extractors
获取 yt-dlp 支持的所有视频平台列表（共 1747+ 个）。
返回示例
{
  "success": true,
  "extractors": ["YouTube", "TikTok", "BiliBili", ...],
  "categories": {
    "国际平台": ["YouTube", "TikTok", "Twitter", "Instagram", "Facebook", "Vimeo", "Dailymotion", "Reddit", "Twitch", "Pornhub"],
    "国内平台": ["Douyin", "BiliBili", "Bilibili", "Weibo", "Xiaohongshu", "Kuaishou"]
  }
}
📋 播放列表与字幕
POST /api/playlist
获取播放列表中的所有视频条目。
请求参数
{
  "url": "https://www.youtube.com/playlist?list=..."
}
返回示例
{
  "success": true,
  "entries": [
    {
      "id": "xxx",
      "title": "视频1",
      "url": "https://www.youtube.com/watch?v=xxx",
      "duration": 120
    },
    {
      "id": "yyy",
      "title": "视频2",
      "url": "https://www.youtube.com/watch?v=yyy",
      "duration": 180
    }
  ]
}
POST /api/subtitles
获取视频可用的字幕/字幕轨道列表。
请求参数
{
  "url": "https://www.youtube.com/watch?v=xxx"
}
返回示例
{
  "success": true,
  "subtitles": [
    {"lang": "en", "formats": ["vtt", "srt", "json3"]},
    {"lang": "zh-Hans", "formats": ["vtt", "srt"]},
    {"lang": "zh-Hant", "formats": ["vtt"]}
  ]
}
注意：此接口仅返回字幕列表，如需下载字幕请使用 yt-dlp 命令行参数 --write-sub。
📁 文件管理
GET /api/files
列出下载目录中的所有文件，按修改时间倒序。
返回示例
{
  "success": true,
  "files": [
    {
      "name": "视频标题_abc12345.mp4",
      "size": 1024000,
      "mtime": 1782900935.9749565
    }
  ]
}
DELETE /api/files/{filename}
删除指定文件。注意：filename 需要 URL 编码。
请求示例
DELETE /api/files/%E8%A7%86%E9%A2%91%E6%A0%87%E9%A2%98_abc12345.mp4
返回示例
{
  "success": true
}
如果文件不存在，返回 404 和 {"success": false, "error": "File not found"}
GET /download/{filename}
直接下载文件，浏览器会触发下载对话框。
请求示例
GET /download/%E8%A7%86%E9%A2%91%E6%A0%87%E9%A2%98_abc12345.mp4
返回文件流，Content-Type 为 application/octet-stream。
GET /api/status
获取服务运行状态和统计信息。
返回示例
{
  "status": "running",
  "files": 5,
  "max_files": 50,
  "download_dir": "/opt/x-download/downloads"
}
GET /api/i18n?lang=zh
获取前端界面所需的翻译文本。
参数
lang: zh | en （默认 zh）
GET 前端调用示例（JavaScript）
使用 JavaScript 调用 API 的完整示例：
// 1. 获取视频信息
async function getVideoInfo(url) {
  const res = await fetch('http://77.73.14.88:8080/api/info', {
    method: 'POST',
    headers: {'Content-Type': 'application/json'},
    body: JSON.stringify({url})
  });
  return await res.json();
}

// 2. 搜索视频
async function searchVideos(query) {
  const res = await fetch('http://77.73.14.88:8080/api/search', {
    method: 'POST',
    headers: {'Content-Type': 'application/json'},
    body: JSON.stringify({query})
  });
  return await res.json();
}

// 3. 下载视频
async function downloadVideo(url, formatId) {
  const res = await fetch('http://77.73.14.88:8080/api/download', {
    method: 'POST',
    headers: {'Content-Type': 'application/json'},
    body: JSON.stringify({url, format: formatId})
  });
  return await res.json();
}

// 4. 下载音频
async function downloadAudio(url) {
  const res = await fetch('http://77.73.14.88:8080/api/audio', {
    method: 'POST',
    headers: {'Content-Type': 'application/json'},
    body: JSON.stringify({url})
  });
  return await res.json();
}

// 5. 获取文件列表
async function getFiles() {
  const res = await fetch('http://77.73.14.88:8080/api/files');
  return await res.json();
}

// 6. 删除文件（filename 需要 encodeURIComponent）
async function deleteFile(filename) {
  const res = await fetch(
    `http://77.73.14.88:8080/api/files/${encodeURIComponent(filename)}`,
    {method: 'DELETE'}
  );
  return await res.json();
}

// 7. 检查链接
async function checkUrl(url) {
  const res = await fetch('http://77.73.14.88:8080/api/check', {
    method: 'POST',
    headers: {'Content-Type': 'application/json'},
    body: JSON.stringify({url})
  });
  return await res.json();
}

// 8. 获取支持平台
async function getExtractors() {
  const res = await fetch('http://77.73.14.88:8080/api/extractors');
  return await res.json();
}
GET 开发注意事项
开发新网页时需要注意：
1. 跨域问题：如果前端不在同一域名下，需要后端支持 CORS
2. 超时设置：下载接口超时 300 秒，获取信息超时 30 秒
3. 文件名编码：涉及文件名的接口需要用 URL 编码
4. 并发限制：yt-dlp 是同步阻塞调用，高并发时会排队
5. 磁盘空间：注意 /opt/x-download/downloads 的磁盘使用情况
6. 平台限制：某些平台需要特殊处理（如抖音/TikTok 需要 Cookie）
7. 搜索功能：默认搜索 YouTube，可通过修改 ytsearch5: 调整结果数量
8. 音频下载：自动转换为 MP3 格式，需要 ffmpeg 支持
